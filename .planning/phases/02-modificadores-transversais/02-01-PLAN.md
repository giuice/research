**Section-by-section specification:**

**Section 1 — Criterios Diagnosticos e de Inclusao:**
- EWGSOP2 FACS algorithm (4 steps: Find/Assess/Confirm/Severity) as the primary diagnostic framework
- Cutoffs: SARC-F >= 4 (screening), grip < 27 kg (M) / < 16 kg (F), ASM/h2 < 7.0 (M) / < 5.5 (F) via DEXA/BIA, gait speed < 0.8 m/s or SPPB <= 8 or TUG > 20 for severity
- Include FACS flow diagram in markdown (as in 02-RESEARCH.md Pattern 2)
- Inclusion: SARC-F >= 4, grip below threshold, age >= 60
- Exclusion: primary neuromuscular disease, active oncologic cachexia
- Note: "Criterios operacionais baseados em EWGSOP2 (2018) — aguardando publicacao dos criterios operacionais do GLIS." Do NOT use GLIS 2024 as operational standard.

**Section 2 — Avaliacao Inicial Mandatoria:**
- TUG + SPPB-BR + Dinamometria Manual (inherited from SHARED-FOUNDATION)
- SARC-F (5 questions, score 0-10)
- Velocidade de marcha 4m (< 0.8 m/s = sarcopenia grave)
- MoCA-BR (cognitive screening — cognitive frailty is frequent in sarcopenia)

**Section 3 — Alinhamento com Diretrizes:**
- EWGSOP2 2019 (primary), GLIS 2024 (contextual only), ESPEN protein guidelines, ACSM resistance training guidelines
- Explicit declaration per guideline with PubMed IDs

**Section 4 — Estrutura de Fases:**
- 3 phases: Fase 1 (Adaptacao/Condicionamento Inicial), Fase 2 (Fortalecimento Progressivo), Fase 3 (Manutencao Funcional)
- Functional entry/exit criteria (not time-based)
- Critical rule: transition by functional criteria only

**Sections 5-7 — Intervencoes por Fase (Modalidade Terapeutica / Terapia Manual / Cinesioterapia):**
- Each phase must have interventions classified under the three categories
- Every exercise with two-column dosage table (Valor Recomendado | Faixa Clinica Aceitavel) and 3 frailty rows (Robusto/Pre-fragil/Fragil)
- Every intervention with Oxford CEBM evidence annotation (Nivel A/B/C + primary citation)
- **CRITICAL — Cross-Protocol Modifier Table:** In the Cinesioterapia subsection of Phase 2 (or Phase 1 if more appropriate), include the frailty-stratified resistance dosing modifier table exactly as specified in 02-RESEARCH.md "Code Examples" section (Template: Cross-Protocol Modifier Label). This table MUST be labeled with the "MODIFICADOR TRANSVERSAL" blockquote. Use the exact parameter values from the research (carga inicial, series, repeticoes, frequencia, duracao minima, sessao, progressao).
- Include multicomponent approach: resistance (primary) + aerobic + balance integrated across phases per 2023-2025 meta-analyses
- Note for Sarcopenia Grave in Fragil tier: reduce initial load by additional 20%

**Section 8 — Monitoramento de Dor:**
- NRS thresholds per phase (inherited format from P-01)
- Differentiation of muscular soreness (DOMS) vs. pathological pain

**Section 9 — Bandeiras Vermelhas e Escalonamento:**
- Red flag table with sarcopenia-specific entries (e.g., unexplained rapid weight loss, falls during exercise, acute muscle pain beyond DOMS)
- Escalation destinations

**Section 10 — Interacoes com Polifarmacia:**
- Start with 6 SHARED-FOUNDATION classes
- ADD sarcopenia-specific classes as in 02-RESEARCH.md template: Estatinas (miotoxicidade), Inibidores de aromatase, Diureticos de alca (hipocalemia), Corticosteroides cronicos (miopatia), Antidepressivos ISRS
- Total must be >= 9 drug class entries
- **Protein-Exercise Coordination subsection** (mandatory): Use the template from 02-RESEARCH.md exactly — protein target >= 1.2 g/kg/dia (PROT-AGE), timing post-exercise, vitamin D target, dietitian referral pathway with 5 criteria. Include scope note: PT does not prescribe diet, only identifies and refers.

**Section 11 — Precaucoes Cardiorrespiratorias:**
- Inherited from SHARED-FOUNDATION; adapt to sarcopenia context (RPE monitoring especially important with beta-blockers)

**Section 12 — Indicadores de Desfecho:**
- Grip strength (MCID: M 5.0-6.5 kg; F 2.7-5.0 kg)
- SPPB-BR (MCID: 0.5-1.0 points)
- Gait speed (MCID: 0.1 m/s)

**Section 13 — Rastreabilidade e Flags de Comorbidade:**

**Section 14 — Referencias por Nivel de Evidencia:**

- Secondary: GLIS 2024 (PMC11210221), drug-sarcopenia (PMC11503558)
